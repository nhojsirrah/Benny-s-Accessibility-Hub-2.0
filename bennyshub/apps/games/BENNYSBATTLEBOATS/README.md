# Benny's Battle Boats

A singleplayer Battleship-style game built with HTML, CSS, and JavaScript.

## Features
- 10x10 grid with row labels A-J and column labels 1-10
- 5 boats to place: 1x2, 2x3, 1x4, 1x5
- Horizontal or vertical placement with a rotate button
- Hidden enemy fleet that is randomly generated each game
- Click enemy squares to attack and sink all enemy ships
- Reset button to start over

## How to run
Open `BENNYSBATTLEBOATS/index.html` in a browser.
